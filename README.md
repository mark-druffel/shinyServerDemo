# shinyServerDemo

A working shiny demo to test deployment

## Installation

``` r
install.packages("pak")
pak::pkg_install("mark-druffel/shinyServerDemo")
```

## Launch Shiny App

Run the app like so:

``` r
library(shinyServerDemo)
run_app("demo")
```
